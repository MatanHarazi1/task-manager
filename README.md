# Task Manager App
אפליקציה פשוטה לניהול משימות יומיומיות המאפשרת הוספה, סימון כבוצע ומחיקה.

## הוראות הרצה לוקאלית:
1. `npm install` - התקנת כל התלויות.
2. `npm run dev` - הרצת הפרויקט בסביבת פיתוח.

## רשימת קומפוננטות:
* **App** - הקומפוננטה הראשית שמנהלת את הסטייט של כל המשימות.
* **TaskInput** - שדה קלט וכפתור להוספת משימה חדשה לרשימה.
* **TaskItem** - מציגה משימה בודדת עם אפשרות למחיקה או סימון כבוצע.

## מגבלות:
* המשימות נשמרות בזיכרון המקומי של הדפדפן (LocalStorage).
* אין תמיכה בעריכת משימה קיימת לאחר הוספתה.
